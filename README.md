# Dashboard

A dashboard design.

![Dashboard Preview](./dashboard.png)
## Features
- **Tech Stack:** React, Tailwind CSS (v4), Vite.
- **Responsive:** Fully responsive layout with a collapsible mobile sidebar.
- **Architecture:** Component-based structure with separated data logic.
- **Fonts:** Used 'Outfit' via Google Fonts to match the geometric design.

## How to Run
1. `npm install`
2. `npm run dev`
